<p>windows下环境安装</p><ul><li>安装node.js</li><li class="ql-indent-1">直接在官网下载-双击-下一步-安装 <a href="https://nodejs.org/en/" target="_blank">https://nodejs.org/en/</a></li><li class="ql-indent-1">命令号输入<span style="background-color: rgb(235, 241, 221);">node -v</span>查看是否安装成功</li><li class="ql-indent-1">(如果报错，删掉当前的node，直接去node官网下载最新版本，更新后问题就可以解决，不要尝试用命令行的方式更新，不好使)</li><li>安装淘宝镜像cnpm <span style="background-color: rgb(235, 241, 221);">npm install cnpm -g--registry=https://registry.npm.taobao.org</span></li><li>默认已经安装好git</li><li>安装vue <span style="background-color: rgb(235, 241, 221);">cnpm install -g vue-cli</span></li><li>创建vue模板 选择一个路径执行<span style="background-color: rgb(235, 241, 221);">vue init webpack my-project</span>（my-project为创建的项目名称） 执行之后会有一系列的初始化设置,完成之后在my-project目录下会生成一些列文件</li><li>进入项目目录<span style="background-color: rgb(235, 241, 221);">cd my-project</span></li><li>安装vue依赖<span style="background-color: rgb(235, 241, 221);">cnpm&nbsp;install</span></li><li>编译工程<span style="background-color: rgb(235, 241, 221);">cnpm run dev</span>(编译之后才可以通过浏览器进行访问localhost:8080)</li></ul><p><br></p>
