<p><strong style="background-color: rgb(238, 240, 244);" class="ql-size-large">windows下解决HTTP Basic: Access denied错误</strong></p><p>windows下git的用户密码输入错误，会出现<span style="background-color: rgb(238, 240, 244); color: rgb(136, 136, 136);">git push 报 HTTP Basic: Access denied 的错误</span></p><p>&nbsp;</p><p><span style="background-color: rgb(238, 240, 244);">解决方案：&nbsp;</span></p><ol><li>下忘记git用户信息配置（这样的缺陷是以后每次都要重新输入用户名和密码）</li></ol><p>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;git config --global credential.helper ""</p><ol><li><span style="background-color: rgb(238, 240, 244);">设置重新输入</span></li></ol><p><span style="background-color: rgb(238, 240, 244);">1. 如果账号密码有变动 用这个命令 git config –system –unset credential.helper 重新输入账号密码 应该就能解决了&nbsp;</span></p><p><span style="background-color: rgb(238, 240, 244);">2. 如果用了第一个命令 还不能解决问题那么 用这个命令：&nbsp;</span></p><p><span style="background-color: rgb(238, 240, 244);">git config –global http.emptyAuth true</span></p><p>&nbsp;</p><p>来自 &lt;<a href="https://blog.csdn.net/u014331288/article/details/79558780" target="_blank">https://blog.csdn.net/u014331288/article/details/79558780</a>&gt; </p><p>&nbsp;</p><p>&nbsp;</p><p>&nbsp;</p><p><span class="ql-size-large">windows下查看git全局配置</span></p><p><span style="background-color: rgb(248, 248, 248);">git config --list --show-origin</span></p><p><span style="background-color: rgb(248, 248, 248);">git config --list --system</span></p><p><span style="background-color: rgb(248, 248, 248);">git config --list --global</span></p><p>&nbsp;</p><p><span class="ql-size-large">进入仓库查看仓库本地配置</span></p><p><span style="background-color: rgb(248, 248, 248);">git config --list --local</span></p><p>&nbsp;</p><p><span class="ql-size-large">修改用户名密码</span></p><p><span style="background-color: rgb(248, 248, 248);">git config --local user.name xxxx</span></p><p><span style="background-color: rgb(248, 248, 248);">git config --local user.email xxxx</span></p><p><span style="background-color: rgb(248, 248, 248);">git config --local user.password xxxx</span></p>
